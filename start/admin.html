板块 #

右下角+号创建

  • 版块名: 你要建设的板块名称
  • 父板块: 你要在哪个板块下建立子板块，点进去选择

设置好后保存

板块详情页

可以更改板块信息

  • 禁用: 停用板块，不在网站上显示该板块

  • 删除板块: 彻底删除板块，谨慎选择

  • 编辑权限: 选择该板块开放给哪些群组用户查看，详情请查看 权限

权限 #

  • registered-users-->注册用户

  • verified-users-->验证用户

  • unverified-users-->未经验证用户

  • Global Moderators-->超级版主

  • banned-users-->禁止用户

  • guests-->游客

  • spiders-->爬虫

    以上均为默认群组，不可删除

因遗留问题，注册用户or未经验证用户，要保持相同

右下角添加群组，可指定某一特定组的权限

也可单独设置某一用户的权限

例:

我只想让验证用户访问这个版块

设置后右下角保存

未经验证邮箱的用户则看不见这个版块，无法访问

注意

guests 游客的权限也应该取消

群组 #

administrators : 超级管理员

Global Moderators: 全局版主

管理员: 管理员可以添加新管理员

注意

管理员默认没有版主权限！需要到每一个新版块开启审核权限
